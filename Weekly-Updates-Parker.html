After joining all of the appropriate communication channels (trello, slack, etc.) provided by Madison, my first task was to get one of the older iterations of our project live on to an actual domain so that it would be easier for our customer to view our progress. To do this, I first set up a new database on a web sever that Zack and I own and then proceeded to populate the database using the sql files given to us by the client from our first meeting. Once this was accomplished, I then created a new FTP directory on our server and purchased a new domain sqstraining.tech. After testing and making sure the new domain had the appropriate DNS settings to route to my new directory, I uploaded all of the current files for the site that were given to us from a previous group into the newly made folder. Finally, I modified the files that were being used to connect to the database so that it would connect properly with the one I made on our server (changing the db name, user, password, and host). So now our project is currently live and viewable at this location.
