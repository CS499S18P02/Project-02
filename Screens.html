This is the homepage of the website. Here you can sign in or sign up for an account


Here we can sign in with an already created account.


The above page is used for creating a user account. The fields on this page are required and have error messages if the input data is incorrect or if the passwords do not match. After this page is submitted, an account will be created associated with this information.


After the user creates their account this registration page will start asking for profile information associated with the user. This page shows off some HTML functionalities that make the registration process easier. These include radio buttons, a dropdown menu, up and down arrows to increment numbers by 1, and a calendar to make selection of the date easier.


This page allows us to add or remove software skills associated with the users profile.


This page allows us to add or remove hardware skills associated with the users profilet


This page is presented immediately after registration and is also accessible through the navigation bar at the top of the website. Here we can see the information input during registration and we can also edit the profile information here.


Here is just a basic homepage for the user. The user can add a phone number for text alerts on this page and this also has error handling associated with it.


This page gives the functionality to the admin to add groups of users together for functions later. They can also create and remove groups as well as promote or demote users in those groups.


Here the admin can see all of the accounts created. They can also see the users info and progress of the users profile. The Super Admin can edit any of this information, except their own, and admins can edit the information of the user accounts below their level. You can also export this information to an excel file on this page using the export button.


This page is used for adding and removing software and hardware skills from the skills bank.


This page is used for adding and removing testing features of the website. These features affect and change the website in a way that is used for training of employees for the customer.


This is also a part of the admin features page. This allows the admin to assign certain features, from the list of features, to groups or users.
