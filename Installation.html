Installation


Download Project

  • Visit this Github Page
  • Click the green "Clone or download" button
  • Click "Download ZIP"
  • Locate the zip file and open it
  • Drag the "SQSTraining-master" folder to your Desktop

Upload Project

  • Connect, using Filezilla, to the FTP server that you wish to upload to
  • Drag the "SQSTraining-master" folder from your Desktop over to the root directory of your server
  • Rename the file accordingly based on a purchased domain
  • If using XAMPP or something similar, simply add the project folder to your htdocs folder

Install Database

  • Connect to the appropriate phpmyadmin for your server
  • Add a new database called "wildnqhi_cs499"
  • Locate the "wildnqhi_cs499.sql" file in the SQSTraining-master
  • Navigate to the "wildnqhi_cs499" database and click the SQL tab
  • Copy/Paste the content from the SQL file into the text box
  • Click "Go"

Success! Your project should be ready to go!
