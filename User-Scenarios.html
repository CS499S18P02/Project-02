User Scenarios


User Scenario A: Access site and Sign up for an account

  • User goes to www.sqstraining.tech
  • User clicks on sign up from the main page.
  • User fills out information on the page that pops up to create your account
  • The user now has an account and can edit the information on his/her profile page which can be accessed through the navigation bar.

User Scenario B: Access site and Sign in to account.

  • User logs in at www.sqstraining.tech
  • User clicks on sign in from the main page.
  • User inputs the information used in their sign up process to sign into account.
  • If the user never made an account then they must refer to Scenario A.

User Scenario C: What generic user can do

  • Generic user can sign up for alerts from the home page.
  • Generic user can see the groups they are included in by going to the Groups page.
  • Generic user can go to the customers about page from their website by clicking on the SQS option on the navigation bar.
  • Generic user can log out from the Log Out option on the navigation bar.

User Scenario D: Admin accessing and editing groups

  • Admin must log on and then navigate to the Groups page on their navigation bar.
  • Admin may then create a group by clicking the add group option and making a name.
  • Admin can then add members to that group or any of the other already created groups by clicking add user.
  • Admin can also remove any users by clicking on the remove option next to the users name in the group.
  • Admin may also delete a group by clicking on the remove group option above each group.
  • Admin may also promote/demote a user to be a leader of the group by clicking the promote or demote option next to the remove option of each user.

User Scenario E: Admin accessing users and editing user information or exporting user information

  • Go to the Admin dropdown menu on the navigation bar.
  • Select Users from the dropdown menu.
  • To edit a users information, click on the pencil icon next to the users account.
  • To delete a users account, click on the trash icon next to the users account.
  • To export all user data, click the export option at the top of the users table.

User Scenario F: Admin editing skills bank

  • Go to the Admin dropdown menu on the navigation bar.
  • Select Skills from the dropdown menu.
  • To add a skill, type the skill in the text box below the skills bank and select Add.
  • To remove a skill, Select the skill from the skills bank and then click the Remove option next to the test box below the skills bank.

User Scenario G: Admin adding a feature or assigning a feature to a user/group

  • Go to the Admin dropdown menu on the navigation bar.
  • Select Features from the dropdown menu.
  • To add a feature, select the Add Features option and input the information for the feature and upload the feature to the pop-up field.
  • To add a feature to a group, click either Add Assignment options at the bottom of the page and select the feature and the group to assign it to from the pop-up.
  • To add a feature to a user, click either Add Assignment options at the bottom of the page and select the feature and the user to assign it to from the pop-up.
  • To unassign a feature from a user/group, click on the Unassign option next to the user or group listed with that feature.
